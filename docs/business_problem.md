# Business Problem

Manufacturing companies must continuously balance future production demand against available manufacturing capacity.

Insufficient capacity may create production delays, while excess capacity may lead to resource underutilization.

The objective of this project is to develop an analytical solution capable of comparing Capacity versus Workload across a 12-month planning horizon, enabling planners to identify bottlenecks, overload situations and available capacity.

# Main Questions

- Do we have sufficient capacity to support future demand?
- Which work centers are overloaded?
- Which periods present capacity shortages?
- What is the utilization rate of each work center?
- What is the expected capacity gap over time?

# Expected Deliverables

- PostgreSQL Database
- Star Schema Data Model
- Analytical SQL Views
- Power BI Dashboard
- Executive Capacity Planning KPIs
